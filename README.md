# Super Smart Study Assistant (SSS)

## Overview

Super Smart Study Assistant (SSS) is an AI-powered educational platform designed to help students organize their studies, generate quizzes from notes, collaborate with peers, 
and improve learning outcomes through interactive design.

## Features

### 📚 Notes Management

* Create and organize study notes
* Associate notes with specific courses
* Store and retrieve notes from the database

### 🧠 AI Quiz Generator

* Generate multiple-choice quizzes from saved notes
* Adjustable difficulty levels (Easy, Medium, Hard)
* Automatic scoring and performance tracking

### 💡 Personalized Learning Support

* AI-generated explanations for incorrect answers
* Adaptive follow-up questions based on mistakes
* Difficulty adjustment recommendations
* Translation Tool for many languages

### 💬 Neuron Network Community

* Ask study-related questions
* Answer questions from other students
* Mark accepted answers
* Filter and search discussions by course

### 📈 Flash Card Generation

* Generates study-related flash cards
* Interactive design
* AI powered

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/Smart-Study-AI.git
```

2. Open the backend project, add API key and run:


*appsettings.json
Add the Gemini API key provided by the Super Smart AI team

```bash
cd backend\SmartStudyAI.Backend
dotnet restore
dotnet run
```

3. Open the frontend in your browser.
